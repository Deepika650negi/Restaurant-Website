# Royal Spice - Restaurant Website

**Royal Spice** is a modern, responsive restaurant website built with **ReactJS** and **Bootstrap 5**.  
It showcases the restaurantâ€™s menu, allows users to search and filter dishes, view ingredients, and add items to a cart for order.

---

## Features

- **Responsive Hero Slider** with beautiful restaurant images  
- **Menu Section** with search, filters (Veg/Non-Veg), and categories  
- **Add to Cart** functionality with quantity management  
- **About, Contact, and Book Table pages**  
- **Floating WhatsApp button** for quick orders  
- **Cart slider** to view selected items and total price  
- Fully styled with **Bootstrap** and custom CSS  

---

## Technologies Used

- **ReactJS**  
- **React Router**  
- **Bootstrap 5**  
- **HTML, CSS, JavaScript**
